# Itinera AI

Itinera AI is your personal AI-powered travel assistant. It creates tailored travel itineraries, helps with booking reservations, and provides real-time travel advice through an intuitive chat interface.

## Features

- **AI-Powered Itineraries**: Get custom travel plans based on your preferences.
- **Chat Interface**: Interact with your assistant via text or voice.
- **Reservation Management**: Keep track of your bookings in one place.
- **Interactive Maps**: Visualize your trip with integrated maps.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **AI**: Vercel AI SDK
- **Components**: Radix UI
